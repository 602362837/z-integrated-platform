#### 3.2.2 树形查询实现

```java
@Service
public class OrgServiceImpl implements IOrgService {

    @Autowired
    private OrgMapper orgMapper;

    /**
     * 获取组织树形列表
     */
    @Override
    public List<OrgVO> getOrgTree() {
        // 1. 查询所有未删除的组织
        List<Org> allOrgs = orgMapper.selectList(
            new QueryWrapper<Org>().orderByAsc("sort_order")
        );

        // 2. 转换为VO并构建树形结构
        return buildTree(allOrgs);
    }

    /**
     * 递归构建树形结构
     */
    private List<OrgVO> buildTree(List<Org> orgs) {
        return orgs.stream()
            .filter(org -> org.getParentId() == 0)
            .map(org -> convertToVO(org, orgs))
            .collect(Collectors.toList());
    }

    private OrgVO convertToVO(Org org, List<Org> allOrgs) {
        OrgVO vo = BeanUtil.copyProperties(org, OrgVO.class);

        // 查找子组织
        List<Org> children = allOrgs.stream()
            .filter(o -> o.getParentId().equals(org.getId()))
            .collect(Collectors.toList());

        if (!children.isEmpty()) {
            vo.setChildren(children.stream()
                .map(child -> convertToVO(child, allOrgs))
                .collect(Collectors.toList()));
        }

        return vo;
    }

    /**
     * 删除组织 - 检查是否有子组织或用户
     */
    @Override
    public void deleteOrg(Long id) {
        // 1. 检查是否有子组织
        long childCount = orgMapper.selectCount(
            new QueryWrapper<Org>().eq("parent_id", id)
        );
        if (childCount > 0) {
            throw new BusinessException("该组织下存在子组织，无法删除");
        }

        // 2. 检查是否有用户
        long userCount = platformUserMapper.selectCount(
            new QueryWrapper<PlatformUser>().eq("org_id", id)
        );
        if (userCount > 0) {
            throw new BusinessException("该组织下存在用户，无法删除");
        }

        // 3. 执行逻辑删除
        orgMapper.deleteById(id);
    }
}
```

#### 3.3.2 用户创建流程

```java
@Service
public class PlatformUserServiceImpl implements IPlatformUserService {

    @Autowired
    private PlatformUserMapper platformUserMapper;

    @Override
    public void createUser(PlatformUserCreateRequest request) {
        // 1. 生成用户code
        String userCode = generateUserCode(request);

        // 2. 校验code唯一性
        PlatformUser existUser = platformUserMapper.selectOne(
            new QueryWrapper<PlatformUser>().eq("code", userCode)
        );
        if (existUser != null) {
            throw new BusinessException("用户编码已存在");
        }

        // 3. 创建用户实体
        PlatformUser user = new PlatformUser();
        user.setCode(userCode);
        user.setUsername(request.getUsername());
        user.setNickname(request.getNickname());
        user.setPassword(passwordEncoder.encode(request.getPassword()));
        user.setEmail(request.getEmail());
        user.setPhone(request.getPhone());
        user.setOrgId(request.getOrgId());
        user.setStatus(StatusEnum.ENABLED.getCode());

        platformUserMapper.insert(user);
    }

    /**
     * 生成用户code
     * - 纯平台用户: PLAT_{UUID}
     * - 应用用户: {appCode}_{userCode}
     */
    private String generateUserCode(PlatformUserCreateRequest request) {
        if (StringUtils.hasText(request.getAppCode())) {
            return request.getAppCode() + "_" + request.getUserCode();
        }
        return "PLAT_" + UUID.randomUUID().toString().replace("-", "");
    }
}
```

### 3.4 应用用户管理模块

#### 3.4.1 用户同步流程

```java
@Service
public class AppUserServiceImpl implements IAppUserService {

    @Autowired
    private AppUserMapper appUserMapper;

    @Autowired
    private PlatformUserMapper platformUserMapper;

    @Autowired
    private IPlatformUserService platformUserService;

    /**
     * 同步应用用户
     */
    @Override
    public void syncAppUsers(String appCode, List<AppUserSyncRequest> users) {
        for (AppUserSyncRequest userRequest : users) {
            syncSingleUser(appCode, userRequest);
        }
    }

    /**
     * 同步单个用户
     */
    private void syncSingleUser(String appCode, AppUserSyncRequest userRequest) {
        // 1. 检查应用用户是否已存在
        AppUser existAppUser = appUserMapper.selectOne(
            new QueryWrapper<AppUser>()
                .eq("app_code", appCode)
                .eq("user_code", userRequest.getUserCode())
        );

        if (existAppUser != null) {
            // 更新应用用户信息
            updateAppUser(existAppUser, userRequest);
        } else {
            // 创建新的应用用户并映射平台用户
            createAndMapAppUser(appCode, userRequest);
        }
    }

    /**
     * 创建应用用户并映射平台用户
     */
    private void createAndMapAppUser(String appCode, AppUserSyncRequest userRequest) {
        // 1. 生成平台用户code: {appCode}_{userCode}
        String platformUserCode = appCode + "_" + userRequest.getUserCode();

        // 2. 查找或创建平台用户
        PlatformUser platformUser = platformUserMapper.selectOne(
            new QueryWrapper<PlatformUser>().eq("code", platformUserCode)
        );

        if (platformUser == null) {
            // 创建新的平台用户
            PlatformUserCreateRequest createRequest = new PlatformUserCreateRequest();
            createRequest.setAppCode(appCode);
            createRequest.setUserCode(userRequest.getUserCode());
            createRequest.setUsername(userRequest.getUsername());
            createRequest.setNickname(userRequest.getNickname());
            createRequest.setEmail(userRequest.getEmail());
            createRequest.setPhone(userRequest.getPhone());
            createRequest.setPassword(UUID.randomUUID().toString()); // 随机密码

            Long userId = platformUserService.createUser(createRequest);
            platformUser = platformUserMapper.selectById(userId);
        }

        // 3. 创建应用用户并建立映射
        AppUser appUser = new AppUser();
        appUser.setAppCode(appCode);
        appUser.setUserCode(userRequest.getUserCode());
        appUser.setUsername(userRequest.getUsername());
        appUser.setNickname(userRequest.getNickname());
        appUser.setEmail(userRequest.getEmail());
        appUser.setPhone(userRequest.getPhone());
        appUser.setPlatformUserId(platformUser.getId());
        appUser.setSyncTime(LocalDateTime.now());

        appUserMapper.insert(appUser);
    }
}
```

## 5. 关键技术点说明

### 5.1 逻辑删除实现

使用 MyBatis-Plus 的 `@TableLogic` 注解实现逻辑删除：

```java
@Data
@TableName("t_app")
public class App {
    @TableLogic
    private Integer deleted;
}
```

配置逻辑删除值：

```yaml
mybatis-plus:
  global-config:
    db-config:
      logic-delete-field: deleted
      logic-delete-value: 1
      logic-not-delete-value: 0
```

### 5.2 树形数据处理

**方案选择**：邻接表模型（Adjacency List）

| 方案 | 优点 | 缺点 | 适用场景 |
|------|------|------|----------|
| 邻接表 | 简单、易维护 | 查询效率低（递归） | 数据量<10000 |
| 物化路径 | 查询效率高 | 更新复杂 | 读多写少 |
| 嵌套集 | 树形查询高效 | 更新复杂 | 读多写少 |
| 闭包表 | 通用性强 | 维护成本高 | 复杂树形操作 |

**本系统选择邻接表**，因为：
- 组织层级不会太深（通常<5层）
- 数据量预估在千级别
- 实现简单，维护成本低

### 5.3 密码加密策略

使用 BCrypt 加密，特点：
- 自动盐值加密
- 单向加密，不可逆
- 每次加密结果不同（因盐值不同）
- 验证时自动提取盐值

```java
// Spring Security BCryptPasswordEncoder
@Bean
public PasswordEncoder passwordEncoder() {
    return new PasswordEncoder();
}
```

### 5.4 用户映射规则实现

```
┌─────────────────────────────────────────────────────────────────┐
│                     用户映射规则                                   │
├─────────────────────────────────────────────────────────────────┤

```

### 5.5 分页与排序

前端使用 Element Plus 的 `el-pagination` 组件：

```vue
<template>
  <el-pagination
    v-model:current-page="query.pageNum"
    v-model:page-size="query.pageSize"
    :page-sizes="[10, 20, 50, 100]"
    :total="total"
    layout="total, sizes, prev, pager, next, jumper"
    @size-change="handleSizeChange"
    @current-change="handleCurrentChange"
  />
</template>
```

后端使用 MyBatis-Plus 的 `IPage`：

```java
@Override
public PageResult<AppVO> getAppPage(AppQuery query) {
    IPage<App> page = new Page<>(query.getPageNum(), query.getPageSize());
    QueryWrapper<App> wrapper = new QueryWrapper<>();

    if (StringUtils.hasText(query.getAppName())) {
        wrapper.like("app_name", query.getAppName());
    }

    IPage<App> result = appMapper.selectPage(page, wrapper);
    return PageResult.of(result, AppVO.class);
}
```

### 5.6 统一响应格式

```java
/**
 * 统一响应结果
 */
@Data
public class Result<T> {
    private Integer code;
    private String message;
    private T data;

    public static <T> Result<T> success(T data) {
        Result<T> result = new Result<>();
        result.setCode(200);
        result.setMessage("success");
        result.setData(data);
        return result;
    }

    public static <T> Result<T> error(String message) {
        Result<T> result = new Result<>();
        result.setCode(500);
        result.setMessage(message);
        return result;
    }
}
```
